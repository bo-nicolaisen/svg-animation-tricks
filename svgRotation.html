Rotating a svg element around its center

the problem

just rotating with this code

    #rotWrong {
    animation: myRotation 1s infinite linear;
    }

    @keyframes myRotation {
    99% { transform: rotate(359deg);}
    }

solution

use this in your CSS

#rotRight {
transform-box: fill-box;
transform-origin: center;


animation: myRotation 1s infinite linear;
}

@keyframes myRotation {
99% {transform: rotate(359deg);}
}
