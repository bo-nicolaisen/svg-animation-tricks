Rotating a svg element around its center

the problem

just rotating with this code

    #rotWrong {
    animation: myRotation 1s infinite linear;
    }

    @keyframes myRotation {
    99% { transform: rotate(359deg);}
    }

solution

use "transform-box: fill-box" in your CSS to set the bounding box of the svg element, and then use transform-origin to set rotation origin.

-læs om transform-box her- -læs om transform-origin her- 

#rotRight {
transform-box: fill-box;
transform-origin: center;


animation: myRotation 1s infinite linear;
}

@keyframes myRotation {
99% {transform: rotate(359deg);}
}
